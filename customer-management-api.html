Customer Management API

A production-ready REST API built with Spring Boot for full CRUD management of customer data, powered by Jackson for JSON handling and an in-memory store.

Spring Boot REST API Jackson Java 17 CRUD

Project Overview

This project demonstrates how to build a clean, well-structured RESTful API with Spring Boot. It covers the full lifecycle of a customer resource — create, read, update, and delete — with proper HTTP status codes and JSON request/response bodies.

Jackson is configured globally via an ObjectMapper bean to handle snake_case field naming, null exclusion, and Java 8 date/time serialization out of the box.

The in-memory store uses a ConcurrentHashMap to keep the project self-contained with no database dependency, making it easy to run and explore locally.

  • Define the ModelJava 17 record Customer with Jackson annotations for JSON field mapping.
  • Build the RepositoryThread-safe in-memory store with ConcurrentHashMap and an atomic ID counter.
  • Expose REST EndpointsController maps HTTP verbs to CRUD operations and returns proper status codes.
  • Handle Errors Globally@RestControllerAdvice catches not-found and validation errors with structured responses.

Code Highlights

Customer Model (Java 17 Record)

Java 17 
@JsonNaming(PropertyNamingStrategies.SnakeCaseStrategy.class)
public record Customer(
    Long   id,
    @NotBlank String firstName,
    @NotBlank String lastName,
    @Email @NotBlank String email,
    String phone
) {
    // Factory method used when creating a new customer (ID assigned by repo)
    public static Customer withId(Long id, Customer src) {
        return new Customer(id, src.firstName(), src.lastName(), src.email(), src.phone());
    }
}

REST Controller

Java 17 
@RestController
@RequestMapping("/api/customers")
@RequiredArgsConstructor
public class CustomerController {

    private final CustomerService service;

    @GetMapping
    public List<Customer> getAll() { return service.findAll(); }

    @GetMapping("/{id}")
    public Customer getOne(@PathVariable Long id) { return service.findById(id); }

    @PostMapping
    public ResponseEntity<Customer> create(@RequestBody @Valid Customer body) {
        return ResponseEntity.status(HttpStatus.CREATED).body(service.create(body));
    }

    @PutMapping("/{id}")
    public Customer update(@PathVariable Long id, @RequestBody @Valid Customer body) {
        return service.update(id, body);
    }

    @DeleteMapping("/{id}")
    @ResponseStatus(HttpStatus.NO_CONTENT)
    public void delete(@PathVariable Long id) { service.delete(id); }
}

In-Memory Repository

Java 17 
@Repository
public class CustomerRepository {

    private final Map<Long, Customer> store = new ConcurrentHashMap<>();
    private final AtomicLong counter = new AtomicLong(1);

    public List<Customer> findAll() { return new ArrayList<>(store.values()); }

    public Optional<Customer> findById(Long id) { return Optional.ofNullable(store.get(id)); }

    public Customer save(Customer c) {
        Long id = c.id() != null ? c.id() : counter.getAndIncrement();
        Customer saved = Customer.withId(id, c);
        store.put(id, saved);
        return saved;
    }

    public boolean deleteById(Long id) { return store.remove(id) != null; }
}
